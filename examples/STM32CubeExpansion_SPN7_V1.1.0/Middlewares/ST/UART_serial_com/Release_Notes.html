Release Note for UART communication for X-CUBE-SPN7 firmware

Copyright 2015 STMicroelectronics

 


UART_serial_com folder: the UART_serial_com folder (\Middlewares\UART_serial_com\) contains the UART library for communication with external PC terminal, for instance hyper-terminal or equivalent. It is possible to send application command to the 6Step control algorithm (start motor, stop motor, set speed, etc..) or get information about the motor speed, control status etc.
