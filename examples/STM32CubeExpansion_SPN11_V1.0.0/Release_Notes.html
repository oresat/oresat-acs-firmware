Release Notes for X-CUBE-SPN11

Software for STM32, Expansion for STM32Cube

Copyright 2016 STMicroelectronics

 


The X-CUBE-SPN11 is an expansion for STM32Cube. This software runs on the STM32 and provides a complete management of the STSPIN230 for control of low voltage three phase brushless motors. The software comes with an example of implementation of the driving of one low voltage 3 phase brushless motor, with BEMF sensing. The example is preconigured for NUCLEO-F401RE connected to one X-NUCLEO-IHM11M1 expansion board, but can be easily ported to other STM32 series thanks to STM32CUBE. The package contains a user interface layer enabling real-time transmission of data to a PC through the terminal.


X-CUBE-SPN11 software features:

  • The package includes an application example to drive a low voltage three phase brushless motor, managing a X-NUCLEO-IHM11M1 board plugged on top of a NUCLEO-F401RE board.
  • Timers managing to generate step clock and voltage reference.
  • Managing min and max speed, direction etc. in device parameters structure.
  • GPIOs, PWMs and IRQs configuration.
  • API function available to send any application command to the motor driver.
  • User interface utility based on PC terminal to control the motor in Comm mode.
  • Speed control by potenziometer.
  • Motor control by user botton.
  • Easy portability across different MCU families, thanks to STM32Cube.
  • Free, user-friendly license terms.

  • The drivers abstract low-level details of the hardware and allow the middleware components and applications to access the STSPIN230 device in a hardware independent fashion.
  • The applications provide examples of how to use the APIs.
